
**Prompt（提示词）** 是在使用 AI（特别是像 OpenAI 开发的 ChatGPT 这类模型）时，**你输入给 AI 的指令、问题或描述**。
简单说：**Prompt = 你对 AI 说的话**。

---

## 1️⃣ 最简单理解

可以把 AI 想成一个“超级自动补全机器”。

你给它一句话开头（Prompt），
它根据这个提示继续生成内容。

例子：

| Prompt          | AI输出       |
| --------------- | ---------- |
| `解释什么是机器学习`     | AI开始解释机器学习 |
| `写一个Python冒泡排序` | AI生成代码     |
| `帮我总结这篇文章`      | AI生成摘要     |

所以：

> **Prompt 决定 AI 输出什么内容。**

---

## 2️⃣ Prompt 在 AI 里的作用

在 **大语言模型（LLM）** 里，所有任务其实都是通过 Prompt 实现的：

| 任务   | Prompt 示例                |
| ---- | ------------------------ |
| 翻译   | `把这句话翻译成英文：你好`           |
| 写代码  | `写一个 Python HTTP server` |
| 解释概念 | `什么是逻辑回归？`               |
| 生成图片 | `生成一张赛博朋克城市夜景`           |
| 数据分析 | `解释这段代码的时间复杂度`           |

所以 AI 本质是：

```
Prompt  →  AI模型 →  输出结果
```

---

## 3️⃣ Prompt 在机器学习里的原始含义

在 AI 领域，Prompt 原本意思是：

**给模型的输入上下文（context）**

例如：

```
Prompt:
Q: 1+1=?
A:
```

模型继续生成：

```
2
```

---

## 4️⃣ Prompt Engineering（提示工程）

因为 **Prompt 会影响 AI 输出质量**，
所以出现一个新领域：

**Prompt Engineering（提示工程）**

意思是：

> **设计更好的 Prompt，让 AI 输出更好的结果。**

例如：

普通 Prompt：

```
解释机器学习
```

优化 Prompt：

```
用通俗语言向一个初学者解释什么是机器学习，
并给出生活中的例子。
```

第二个输出会更好。

---

## 5️⃣ Prompt 的常见结构

好的 Prompt 通常包含：

```
角色 + 任务 + 限制 + 输出格式
```

例如：

```
你是一名机器学习老师，
用简单语言解释什么是逻辑回归，
给出生活例子，
用 Markdown 列表输出。
```

---

## 6️⃣ 一个非常形象的理解

可以这样理解：

```
AI模型 = 大脑
Prompt = 问题
输出 = 回答
```

所以：

> **Prompt 就是你给 AI 的“问题/指令”。**
