### 有监督学习 与 无监督学习
* 有监督学习：通过有标记数据进行训练，预测新数据的标签或值。<br/>
* 无监督学习：在没有明确标签的情况下发现数据中的模式或结构。 <br/>

3、 🧠 1️⃣ 什么是「过拟合」(Overfitting)？
一句话：
> 模型把训练数据“背下来”，但不会举一反三。
### 比喻
学生死记硬背题库：
* 题目一样 → 100%
* 换一套题 → 掉到 70%
这就是过拟合。
在代码里典型现象：
```text
训练准确率 = 100%
测试准确率 = 很低
```
4、 🧠 2️⃣ 什么是「欠拟合」(Underfitting)？
一句话：
> 模型太简单，连训练数据都学不好。
比喻：
学生根本没学会知识：
* 原题都做错
* 新题更做错
表现是：
```text
训练准确率 = 低
测试准确率 = 也低
```
5、 🧠 3️⃣ 什么是「泛化能力」(Generalization)？
一句话：
> 模型对“新数据”的表现能力。
真正重要的不是：
```text
训练集表现
```
而是：
```text
测试集表现
```
测试集好 → 泛化能力强
测试集差 → 泛化能力弱
# 🧠 4️⃣ 什么是「逻辑回归」？
它是一个：
> 线性分类模型
简单理解：
它会画一条“直线”或“平面”，把不同类别分开。
优点：
* 简单
* 稳定
* 不容易过拟合
缺点：
* 只能处理“比较规整”的数据
---
# 🧠 5️⃣ 什么是「决策树」？
它像这样思考：
```text
如果 花瓣长度 > 2
    就是 A
否则
    是 B
```
一层层分下去。
优点：
* 好理解
* 灵活
缺点：
* 很容易过拟合
* 容易“记住数据”
---
# 🧠 6️⃣ 什么是「KNN」？
全名：
K-Nearest Neighbors（K最近邻）
思想是：
> 看离我最近的 K 个人是谁，我就属于谁那一类。
---
## 为什么 K=1 容易过拟合？
当：
```python
K = 1
```
模型只看“最近的一个样本”。
在训练数据里：
每个样本离自己最近
→ 所以训练准确率 100%
但换新数据：
很容易判断错。

# 另: “过拟合  欠拟合” 的异同
![image](https://github.com/shaoting0730/other-learn/blob/master/AI%E5%85%A5%E9%97%A8/%E5%85%A5%E9%97%A8%E4%B8%AD/%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0Machine%20Learning(ML)/%E2%80%9C%E8%BF%87%E6%8B%9F%E5%90%88%20vs%20%E6%AC%A0%E6%8B%9F%E5%90%88%E2%80%9D%E7%9A%84%E7%9B%B4%E8%A7%82%E5%9B%BE.png) <br/>

你看到两条线：

### 📉 一条一直往下的线

这是 **训练误差**

模型越复杂：

* 越容易把训练数据记住
* 训练误差越来越低

---

### 📈 一条“U形”的线

这是 **测试误差**

左边：

* 模型太简单
* 连规律都没学会
* 👉 欠拟合

中间最低点：

* 复杂度刚刚好
* 👉 泛化能力最强（最佳模型）

右边：

* 模型太复杂
* 开始记住噪声
* 👉 过拟合

---

# 🔥 用一句话记住

```text
欠拟合：太简单
过拟合：太复杂
最佳点：刚刚好
```

# 🧠 再结合你之前的实验

* 决策树默认深度很大 → 容易跑到右边（过拟合）
* K=1 KNN → 非常靠右（极高复杂度）
* 逻辑回归 → 靠左一点（稳定型模型）

---

# 🎯 真正机器学习的核心

不是追求 100% 准确率。

而是：

> 找到测试误差最低的位置。
