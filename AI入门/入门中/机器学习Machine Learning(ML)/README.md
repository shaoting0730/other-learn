# 机器学习 = 让计算机从数据中“学规律”
## 给数据 + 给答案 → 让机器自己找规则

基本概念:  <br/>
**回归**:  <br/>
“回归”（Regression）这个词来自统计学。在数学和统计学里:用一个函数去拟合数据之间的关系,简单说就是找到一个函数，让输入和输出之间的关系被表达出来。 <br/>
在机器学习里  <br/>
> 回归 = 预测一个连续数值,用一个函数去预测连续数值，并刻画变量之间的关系。 <br/>

比如:  <br/>
预测房价（350万） <br/>
预测温度（26.3°C） <br/>
预测股票价格 <br/>
预测销量（10234件） <br/>
**拟合**:  <br/>
> 用数学函数去尽量接近数据的真实规律。 <br/>

简单说：用数学公式去“模仿”数据的规律。<br/>
拟合不好（欠拟合） <br/>
拟合过头（过拟合） <br/>
**泛化能力（Generalization Ability）**:  <br/>
> 从有限样本中学习到一般规律,如 举一反三 <br/>

过拟合 = 泛化能力差 <br/>

**核函数（Kernel Function) **:  <br/>
核函数（Kernel Function）是机器学习中用来“隐式地”把数据映射到高维空间，并在那个空间里计算内积的一种函数。
> 👉 核函数 = 不用真的升维，也能在高维空间里做线性运算的技巧。 <br/>

---

### 有监督学习 与 无监督学习
* 有监督学习：通过有标记数据进行训练，预测新数据的标签或值。<br/>
* 无监督学习：在没有明确标签的情况下发现数据中的模式或结构。 <br/> <br/>

**KNN（K-Nearest Neighbors，K最近邻）** 是 Machine Learning 里一种非常直观的算法。很多机器学习课程都会用它来讲 **分类和回归的基本思想**。

一句话理解：

**KNN = 看你“附近”的K个样本是什么，就判断你是什么。**

---

# 一、KNN的直觉理解

假设要判断一种动物是 **猫还是狗**。

我们有训练数据：

| 体重   | 耳朵长度 | 类别 |
| ---- | ---- | -- |
| 4kg  | 5cm  | 猫  |
| 5kg  | 6cm  | 猫  |
| 18kg | 12cm | 狗  |
| 20kg | 13cm | 狗  |

现在有一个新动物：

```
体重 = 6kg
耳朵 = 6cm
```

KNN会做一件事：

**找到最接近它的K个样本。**

例如：

```
K = 3
```

最近的3个：

```
猫
猫
狗
```

多数是：

```
猫
```

所以预测：

```
猫
```

---

# 二、为什么叫“最近邻”

KNN核心思想：

```
物以类聚
```

在数学上就是：

**距离越近 → 越相似**

因此要计算：

```
数据之间的距离
```

最常见：

**欧氏距离**

来自 Euclidean Geometry。

公式：

[
d = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}
]

例如：

```
点A (1,2)
点B (4,6)
```

距离：

```
sqrt((1-4)^2 + (2-6)^2)
```

# 三、KNN算法流程

完整流程：

```
训练数据
   ↓
输入新数据
   ↓
计算与所有数据距离
   ↓
找到最近K个
   ↓
投票
   ↓
得到结果
```

例如：

```
K=5
最近邻：
猫 猫 狗 猫 狗
```

多数：

```
猫
```

预测：

```
猫
```

# 四、KNN也可以做回归

不仅能分类，还能做回归。

例如预测房价：

最近3个房子：

```
380万
400万
420万
```

预测值：

```
平均值 = 400万
```

所以：

```
KNN回归 = 邻居的平均值
```

---

# 五、K值是什么意思

K是：

```
参考邻居数量
```

例如：

```
K=1
```

只看 **最近一个**。

```
K=5
```

看 **最近五个**。

---

# 六、K值的影响（非常重要）

### K太小

例如

```
K=1
```

问题：

```
容易受噪声影响
```

容易 **过拟合**。

---

### K太大

例如

```
K=50
```

问题：

```
远处的数据也被算进去
```

容易 **欠拟合**。

---

一般经验：

```
K ≈ √N
```

N = 数据量

---

# 七、KNN的特点

优点：

```
非常简单
无需训练
效果稳定
```

缺点：

```
计算量大
数据多时很慢
高维数据效果差
```

因为每次预测都要：

```
和所有数据算距离
```
